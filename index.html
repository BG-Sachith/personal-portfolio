PROJECTS

Key Projects

  • **FieldEx Platform**: Led the development of a dynamic platform for managing field services, where I implemented end-to-end solutions for both frontend and backend systems. This project involved creating complex user interfaces with **Angular** and building scalable backend services using **Spring Boot** and **JHipster**.

  • **Blog**: Developed a full-featured **Blog Platform** where users can read, write, and interact with blog posts. This platform evolved over time with two major versions: Old Version: - Built with **Next.js**, **MongoDB**, and **Redux**. The **MongoDB** database provided flexibility in storing unstructured data for posts and comments, and **Redux** was used to manage application state, such as user authentication and post management, across the platform. - Key features of this version: - User authentication and session management with JWT tokens. - Users could create, read, update, and delete blog posts with comments powered by **MongoDB**. - Frontend built with **Next.js** for SSR/SSG, providing SEO-friendly and performant pages. - Real-time updates using **Redux** to handle post data, user sessions, and dynamic content display. New Version: - Rebuilt with **Next.js**, **MySQL**, and **Prisma**. This updated version improves performance and scalability with a relational **MySQL** database, leveraging **Prisma** as an ORM to manage structured data effectively. - Key improvements: - Data storage has been optimized with a relational database (MySQL), which allows for more complex queries and relationships between posts, users, and comments. - **Prisma** was implemented to interact with the MySQL database, improving query efficiency and ease of database management. - **State management** is now handled with **Redux**, ensuring seamless user experiences for handling authentication, post management, and real-time updates. - The user interface remains highly responsive, built with **Next.js** for fast, SEO-optimized pages with SSR/SSG. This evolution reflects my commitment to optimizing the platform for scalability, performance, and a better user experience.

The platform allows users to access educational resources, tutorials, and peer interactions. Built with **Angular** and **Spring Boot**, the project utilized the latest web development technologies to ensure smooth navigation, data integration, and real-time interactions.
